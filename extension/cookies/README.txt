This folder is reserved for the CEP/CEF browser profile used by the AI Hub mini browser.

cef-profile stores Chromium profile/session data when Adobe CEP honors the
--user-data-dir flag from CSXS/manifest.xml.

cef-cache stores browser cache when Adobe CEP honors the --disk-cache-dir flag.

Do not manually paste third-party cookies here. Let the embedded browser create
and manage session data normally after a user logs in.
